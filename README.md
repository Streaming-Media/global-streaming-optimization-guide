# 全球流媒体（Netflix / YouTube）订阅与画质优化指南

## 📺 Netflix 4K 解锁与版权策略

Netflix 的内容库随 IP 物理归属地实时变化。

* **版权分区**：通过技术手段确保 IP 落在对应地区（如美区、土耳其区），以解锁特定的独占剧集。
* **画质优化**：若无法触发 4K/HDR，请检查链路的 RTT 延迟是否稳定。建议使用支持原生流媒体解锁的节点，以避开 Netflix 对代理服务器的检测。

## 🎵 YouTube Premium 稳定运行规范

* **家庭组安全**：加入家庭组时，建议使用纯净的浏览器环境操作。严禁短时间内频繁切换家庭组区域。
* **免广告策略**：确保您的访问环境能够完全屏蔽底层广告追踪协议，获取纯净的无广告播放体验。

## ⚙️ 底层链路对画质的影响

流媒体播放的稳定性极度依赖于持续的带宽输出。

1. **缓存预取**：高质量的链路能减少视频初始加载时间。
2. **防降级检测**：平台会根据您的实时网络权重调整分辨率。维持一个纯净且稳定的 IP 环境，是防止画质自动降级为 720P 的关键。
